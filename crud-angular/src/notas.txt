EXECUTAR PROJETO
npm run start

git remote show origin

comandos
ng add @angular/material --> adicionar material
ng g m courses --routing --> modulo com roteamento
ng g c  courses/courses --> componente courses dentro do modulo courses
ng g c courses/course-form --> adicionar componente course-form
ng g m shared/app-material --> imports do angular-material em um unico modulo
ng g pipe shared/pipes/category --> pipes transforma valores
ng g resolver courses/guards/course --> cria um resolver
ng update @angular/cli@13.3.9 @angular/core@13.3.9
ng update @angular/core@15 @angular/cli@15
